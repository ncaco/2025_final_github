# 게시판 관리자 메뉴 구조 설계

## 1. 개요
게시판 시스템의 관리자 메뉴 구조를 설계합니다. 효율적인 콘텐츠 관리와 시스템 모니터링을 목표로 합니다.

## 2. 관리자 대시보드

### 2.1 메인 대시보드 위젯
```
📊 시스템 현황
├── 총 게시글 수 | 오늘 게시글 수
├── 총 사용자 수 | 오늘 가입자 수
├── 총 댓글 수 | 오늘 댓글 수
└── 신고 대기 수 | 처리율

📈 실시간 차트
├── 시간별 게시글 작성 추이
├── 게시판별 활동량
├── 사용자 유형 분포
└── 신고 처리 현황

🚨 긴급 알림
├── 미처리 신고
├── 시스템 경고
├── 용량 경고
└── 성능 이슈

📋 최근 활동
├── 최신 게시글
├── 최신 댓글
├── 신고 내역
└── 관리자 작업 로그
```

## 3. 메인 네비게이션 (관리자 패널)

### 3.1 좌측 사이드바 메뉴
```
🎛️ 관리자 패널
├── 📊 대시보드
├── 📋 게시판 관리
├── 📝 콘텐츠 관리
├── 👥 사용자 관리
├── 📈 통계 분석
├── ⚙️ 시스템 설정
└── 📝 작업 로그
```

### 3.2 상단 툴바 (빠른 액션)
```
⚡ 빠른 액션
├── 🚨 신고 처리 (뱃지)
├── 📝 게시글 작성
├── 👤 사용자 검색
├── 🔍 콘텐츠 검색
└── 📊 리포트 생성
```

## 4. 게시판 관리 메뉴

### 4.1 게시판 목록 및 관리
```
📋 게시판 관리
├── 게시판 목록
│   ├── 전체 게시판 조회
│   ├── 게시판 생성
│   ├── 게시판 수정/삭제
│   └── 게시판 순서 변경
│
├── 카테고리 관리
│   ├── 카테고리 생성/수정/삭제
│   ├── 카테고리별 통계
│   └── 카테고리 순서 변경
│
└── 게시판 설정
    ├── 권한 설정 (읽기/쓰기/댓글)
    ├── 첨부파일 설정 (크기/유형 제한)
    ├── 알림 설정
    └── SEO 설정
```

### 4.2 게시판별 상세 관리
```
게시판 상세 관리
├── 기본 정보
│   ├── 게시판 정보 수정
│   ├── 설명 및 규칙 설정
│   └── 대표 이미지 설정
│
├── 권한 관리
│   ├── 읽기 권한 설정
│   ├── 쓰기 권한 설정
│   ├── 댓글 권한 설정
│   └── 관리자 권한 설정
│
├── 기능 설정
│   ├── 태그 사용 여부
│   ├── 첨부파일 허용
│   ├── 익명 글쓰기
│   └── 추천 기능
│
└── 통계 및 모니터링
    ├── 게시판 통계
    ├── 사용자 활동 분석
    ├── 인기 콘텐츠 분석
    └── 트렌드 분석
```
