# 게시판 사용자 메뉴 구조 설계

## 1. 개요
게시판 시스템의 사용자 메뉴 구조를 설계합니다. 직관적이고 사용하기 쉬운 네비게이션 구조를 목표로 합니다.

## 2. 메인 네비게이션 (상단/하단)

### 2.1 데스크톱 상단 메뉴
```
[로고] 게시판 시스템
├── 홈
├── 게시판 목록 ▼
│   ├── 전체 게시판
│   ├── 인기 게시판
│   ├── 내가 팔로우한 게시판
│   └── ──────────
│   ├── 공지사항
│   ├── 자유게시판
│   ├── 질문답변
│   └── 갤러리
├── 검색 🔍
├── 알림 🔔 (뱃지 표시)
└── 사용자 메뉴 ▼
    ├── 내 프로필
    ├── 북마크
    ├── 팔로우 목록
    ├── 설정
    └── 로그아웃
```

### 2.2 모바일 하단 네비게이션
```
🏠 홈    📋 게시판    🔍 검색    🔔 알림    👤 마이페이지
```

## 3. 사이드바 메뉴 (좌측 패널)

### 3.1 게시판별 메뉴
```
📌 게시판 정보
├── 게시판 소개
├── 게시판 규칙
└── 게시판 통계

📂 카테고리
├── 전체
├── 일상
├── 취미
├── 정보
└── [더보기]

🔥 인기 태그
├── #질문
├── #정보공유
├── #일상
└── #추천

👥 팔로워
├── 팔로워 목록
├── 팔로잉 목록
└── 추천 사용자
```

### 3.2 사용자 프로필 메뉴
```
👤 프로필 정보
├── 프로필 수정
├── 아바타 변경
└── 소개글 수정

📊 활동 통계
├── 작성한 글
├── 작성한 댓글
├── 받은 좋아요
└── 팔로워 수

⚙️ 설정
├── 알림 설정
├── 개인정보 설정
├── 테마 설정
└── 언어 설정
```

## 4. 페이지별 메뉴 구조

### 4.1 홈 대시보드
```
🎯 오늘의 추천 게시글
📈 인기 게시글 차트
🆕 최신 게시글
💬 실시간 댓글
🏷️ 인기 태그 클라우드
👥 추천 사용자
```

### 4.2 게시판 목록 페이지
```
📋 게시판 목록
├── 필터: 전체 | 인기 | 팔로우 | 추천
├── 정렬: 이름순 | 게시글수 | 생성일
└── 검색: 게시판명 검색

각 게시판 카드:
├── 게시판명 + 설명
├── 통계: 게시글수 | 오늘글수 | 팔로워수
├── 최근 게시글 미리보기
└── 팔로우/언팔로우 버튼
```

### 4.3 게시글 목록 페이지
```
📝 게시글 목록
├── 필터: 전체 | 공지 | 일반 | 인기
├── 정렬: 최신순 | 조회순 | 추천순 | 댓글순
├── 검색: 제목+내용 검색
└── 태그 필터

작업 버튼:
├── ✏️ 글쓰기
├── 📊 통계 보기
└── 📋 게시판 설정 (관리자만)

각 게시글 아이템:
├── 제목 + 요약
├── 작성자 + 작성일
├── 조회수 | 추천수 | 댓글수
├── 태그 목록
└── 액션: 북마크 | 공유 | 신고
```

### 4.4 게시글 상세 페이지
```
📄 게시글 내용
├── 제목 + 작성자 정보
├── 본문 내용
├── 첨부파일 목록
└── 태그 목록

💬 댓글 영역
├── 댓글 쓰기
├── 댓글 목록 (계층 구조)
├── 댓글 정렬: 최신순 | 추천순
└── 댓글 페이징

🔗 관련 기능
├── 이전글/다음글
├── 비슷한 글 추천
├── 북마크 토글
└── 공유 메뉴

⚡ 빠른 액션
├── ❤️ 좋아요
├── 🔖 북마크
├── 📤 공유
└── 🚨 신고
```

### 4.5 검색 페이지
```
🔍 통합 검색
├── 검색어 입력
├── 검색 범위: 전체 | 제목 | 내용 | 작성자 | 태그
├── 기간 필터: 전체 | 1일 | 1주 | 1개월 | 1년
└── 게시판 필터

📊 검색 결과
├── 게시글 결과
├── 댓글 결과
├── 사용자 결과
└── 태그 결과

🔥 인기 검색어
📝 최근 검색어
💡 검색 제안
```

### 4.6 알림 페이지
```
🔔 알림 센터
├── 필터: 전체 | 읽지않음 | 오늘 | 이번주
├── 정렬: 최신순 | 유형별
└── 모두 읽음 처리

알림 유형별:
├── 💬 댓글 알림
├── ❤️ 좋아요 알림
├── 👥 팔로우 알림
├── 📢 공지사항
└── ⚠️ 시스템 알림

각 알림 아이템:
├── 알림 유형 아이콘
├── 내용 요약
├── 관련 링크
├── 발생 시간
└── 읽음 상태
```

## 5. 컨텍스트 메뉴 (우클릭/롱프레스)

### 5.1 게시글 컨텍스트 메뉴
```
📝 게시글 메뉴
├── 🔗 링크 복사
├── 📤 공유하기
├── 🔖 북마크 토글
├── 📊 통계 보기
├── ✏️ 수정하기 (본인/관리자)
├── 🗑️ 삭제하기 (본인/관리자)
└── 🚨 신고하기
```

### 5.2 댓글 컨텍스트 메뉴
```
💬 댓글 메뉴
├── 🔗 링크 복사
├── ✏️ 수정하기 (본인/관리자)
├── 🗑️ 삭제하기 (본인/관리자)
├── ❤️ 추천하기
└── 🚨 신고하기
```

### 5.3 사용자 컨텍스트 메뉴
```
👤 사용자 메뉴
├── 👀 프로필 보기
├── 💬 메시지 보내기
├── 👥 팔로우 토글
├── 🚫 차단하기
└── 🚨 신고하기
```

## 6. 퀵 액션 메뉴 (플로팅 버튼)

### 6.1 메인 플로팅 버튼
```
➕ 빠른 작성
├── ✏️ 게시글 쓰기
├── 💬 댓글 쓰기 (게시글에서)
├── 📷 사진 업로드
└── 🎥 동영상 업로드
```

### 6.2 페이지별 플로팅 버튼
```
게시판 페이지: 📝 글쓰기
검색 페이지: 🔍 고급 검색
알림 페이지: ✅ 모두 읽음
프로필 페이지: ⚙️ 설정
```

## 7. 브레드크럼 (Breadcrumb)

### 7.1 네비게이션 경로 표시
```
홈 > 게시판 > 자유게시판 > [게시글 제목]
홈 > 사용자 > [사용자명] > 작성한 글
홈 > 검색 > "키워드" > 결과
```

## 8. 반응형 메뉴 설계

### 8.1 태블릿 (768px - 1024px)
- 상단 메뉴 유지
- 하단 네비게이션 추가
- 사이드바 축소 가능

### 8.2 모바일 (320px - 767px)
- 상단 메뉴 → 햄버거 메뉴
- 하단 탭 네비게이션
- 드로어 메뉴 사용
- 터치 친화적 버튼 크기

## 9. 접근성 고려사항

### 9.1 키보드 네비게이션
- Tab 키로 메뉴 이동
- Enter/Space로 메뉴 선택
- Escape로 메뉴 닫기

### 9.2 스크린 리더 지원
- ARIA 레이블 적용
- 의미있는 링크 텍스트
- 포커스 표시

### 9.3 고대비 모드 지원
- 색상 대비 고려
- 아이콘 + 텍스트 조합
- 명확한 상태 표시
