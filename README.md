## 2025_final_github

2025년을 마무리하는 **Next.js 관리자/분석 대시보드** 풀스택 연습용 리포지토리입니다.  
프론트엔드는 `frontend` 디렉터리의 Next.js(App Router) 기반 프로젝트로 구성되어 있습니다.

---

### 프로젝트 구조 개요

- **frontend**: Next.js 16 + TypeScript + Tailwind + shadcn/ui 기반 관리자/분석 대시보드
- **db_structure.md**: PostgreSQL + Prisma 기준 데이터베이스 구조 문서 (싱글 소스 오브 트루스)
- **2025_keywords.md / web_*_2025_trends.md**: 2025년 웹 트렌드/키워드 정리 메모

자세한 프론트엔드 실행/개발 방법은 `frontend/README.md`를 참고하세요.
